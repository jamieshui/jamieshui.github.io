# 1. 如何理解 HTML 语义化？

  • 对人：增加代码的可读性，让代码更容易维护
  • 对机器：对搜索引擎更加友好，有利于SEO（Search Engine Optimization）

# 2. 块级元素和内联元素

考点：块级元素和内联元素有哪些及特点，以及之间怎么相互转换？

# 2.1. 块级元素

  • 常见的块级元素：p、div、form、ul、li、ol、table、h1、h2、h3、h4、h5、h6、dl、dt、dd
  • 块级元素会独自占据一行，可以声明宽和高
  • display:block可以把元素转换为块级元素

# 2.2. 行内元素

  • 常见的行级元素：span、a、img、button、input、select
  • 行内元素会出现在一行，不会另起新行。不能直接声明宽高，如果要声明宽高需要把元素转换为块级或者行内块元素。
  • display:inline把元素转换为行内元素

# 2.3. 行内块元素

  • 不自动换行，可以声明宽高
  • display: inline-block将元素转换为行内块元素
