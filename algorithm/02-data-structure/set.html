# 什么是集合？

  • 一种无序的且唯一的数据结构。

# 集合的应用场景

  • ES6 中的 Set 即为封装好的集合数据结构，可直接使用。

# 🔥 场景一：数组去重

  • 面试常考题型，一般会结合数组扁平化知识点综合考察，比如将 arr = [1, 2, 3, 4, [3, 4, [4, 6]]] 去重：

    function flat(arr, target) { // 数组扁平化
  arr.forEach(item => {
    if (Array.isArray(item)) {
      flat(item, target)
    } else {
      target.push(item)
    }
  })
}

function flatArr(arr) {
  let result = []
  flat(arr, result)
  return result
}

function uniqueArr(arr) {
  return [...new Set(flatArr(arr))] // 数组去重
}

# 场景二：判断某元素是否在集合中

  • 利用Set.has(item)方法。
const arr = [1, 1, 2, 2];

const set = new Set(arr);
const has = set.has(3); // false

# 场景三：求交集

const arr1 = [1, 1, 2, 2];
const arr2 = [2, 3];

const set1 = new Set(arr1);
const set2 = new Set(arr2);
const setRes = new Set([...set1].filter(item => set2.has(item))); // setRes = [2]

# 使用 Set 对象方法

  • add(value) 向集合添加一个新的项。
  • delete(value) 从集合移除一个值。
  • has(value) 如果值在集合中，返回 true，否则返回false。

# Set 常用操作

let mySet = new Set();

# 多种迭代方法

  • for (let item of mySet)

  • for (let item of mySet.keys())

  • for (let item of mySet.values())

# Set 和 Array 互转

  • const myArr = [...mySet]; // 或者 Array.from(mySet)

  • const set = new Set(arr);

# LeetCode 349. 两个数组的交集

  • 题目链接：https://leetcode-cn.com/problems/intersection-of-two-arrays/ (opens new window)

# 题目描述

给定两个数组，编写一个函数来计算它们的交集。

# 解题思路

  • 利用 ES6 的 Set 集合先去重再遍历判断相交元素。

# 实现代码

/**
 * @param {number[]} nums1
 * @param {number[]} nums2
 * @return {number[]}
 */
 var intersection = function(nums1, nums2) {
    return [...new Set(nums1)].filter(item => nums2.includes(item));
};
