# 栈的常见操作

  • push() 入栈，添加一个新元素到栈顶位置。
  • pop() 出栈，移除栈顶的元素，同时返回被移除的元素。
  • peek() 返回栈顶的元素，且不对栈做任何修改。
  • isEmpty() 如果栈里没有任何元素就返回 true ，否则返回 false 。
  • size()返回栈里的元素个数。
  • toString() 将栈结构的内容以字符串的形式返回。

# JS模拟实现栈结构

  • JavaScript中没有栈，但可以用 Array 模拟实现栈的所有功能。
// 栈结构的封装
class Stack {
  constructor() {
    this.items = [];
  }

  // push(item) 压栈操作，往栈里面添加元素
  push(item) {
    this.items.push(item);
  }

  // pop() 出栈操作，从栈中取出元素，并返回取出的那个元素
  pop() {
    return this.items.pop();
  }

  // peek() 查看栈顶元素
  peek() {
    return this.items[this.items.length - 1];
  }

  // isEmpty() 判断栈是否为空
  isEmpty() {
    return this.items.length === 0;
  }

  // size() 获取栈中元素个数
  size() {
    return this.items.length;
  }

  // toString() 返回以字符串形式的栈内元素数据
  toString() {
    let result = "";
    for (let item of this.items) {
      result += item + " ";
    }
    return result;
  }
}

# 场景一：十进制转二进制

  • 将余数依次入栈，然后再出栈，就可以实现余数倒序输出。
// 利用栈结构的特点封装实现十进制转换为二进制的方法
function dec2bin(dec) {
  // new 一个 栈结构，保存余数
  const stack = new Stack();

  // 当不确定循环次数时，使用 while 循环
  while (dec > 0) {
    // 除二取余法
    stack.push(dec % 2); // 获取余数，放入栈中
    dec = Math.floor(dec / 2); // 除数除以二，向下取整
  }

  let binaryString = "";
  // 不断地从栈中取出元素（0 或 1），并拼接到一起。
  while (!stack.isEmpty()) {
    binaryString += stack.pop();
  }

  return binaryString;
}

  • JS解释器使用栈来控制函数的调用顺序，最后调用的函数，最先执行完。

# 引申问题：为什么没有停止条件的递归会造成栈溢出？

  • 比如函数A为递归函数，不断地调用自己，即不停地把相同的函数A压入栈，但由于函数还没有执行完，不会将函数弹出栈，最后造成栈溢出。

# LeetCode 20. 有效的括号

# 题目描述

给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串 s ，判断字符串是否有效。

  • 题目链接：https://leetcode-cn.com/problems/valid-parentheses/ (opens new window)

# 解题思路

  • 左括号入栈，右括号出栈，最后栈空了就是合法的。
  • 利用 Array 模拟实现栈操作。

# 实现代码

/**
 * @param {string} s
 * @return {boolean}
 */
 var isValid = function(s) {
    if (s.length % 2) return false; // 判断奇偶性
    const stack = [];
    for (let x of s) {
        if (x === '(' || x === '[' || x === '{') {
            stack.push(x);
        } else {
            let val = stack.pop();
            if (
                (val === '(' && x === ')') ||
                (val === '[' && x === ']') ||
                (val === '{' && x === '}')
            ) {
                continue;
            }
            else return false;
        }
    }
    return stack.length === 0;
};

# LeetCode 144. 二叉树的前序遍历

  • 题目链接：https://leetcode-cn.com/problems/binary-tree-preorder-traversal/ (opens new window)

# 题目描述

给你二叉树的根节点 root ，返回它节点值的 前序 遍历。

# 解题思路

  • 用栈模拟实现函数递归调用过程，即用栈实现迭代过程。

# 实现代码

/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number[]}
 */
 var preorderTraversal = function(root) {
    if (!root) return [];
    const stack = [root];
    const res = [];
    while (stack.length) {
        const node = stack.pop();
        res.push(node.val);
        if (node.right) stack.push(node.right);
        if (node.left) stack.push(node.left);
    }
    return res;
};
